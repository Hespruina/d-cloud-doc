# PVP规则

服务器PVP相关规则和注意事项。

## 基本规则

### 允许的行为

- 在指定PVP区域进行战斗
- 经双方同意的PVP对决
- 参与活动中的PVP环节
- 自卫反击(在被攻击时)

### 禁止的行为

- 在安全区域(PVE区域)攻击他人
- 组队欺负单个玩家(2v1, 3v1等)
- 利用地形或漏洞击杀玩家
- 击杀新手玩家(游戏时间小于2小时)
- 死亡后继续攻击(守尸)

## PVP区域

### 竞技场

位于服务器西侧,提供正式PVP场所:

- 标准1v1擂台
- 组队战场(2v2, 3v3)
- 自由混战区

### 野外PVP

在部分野外区域允许PVP:

- 边界之地(距离出生点5000格外)
- 下界全区域
- 末地全区域

这些区域不保证安全,建议结伴出行。

## 装备说明

### 竞技场

提供标准装备,包括:

- 钻石套(保护IV)
- 钻石剑(锋利V)
- 弓(力量V)
- 金苹果 x10
- 箭 x64

### 野外PVP

使用自带装备,限制:

- 禁止使用附魔超过服务器上限的装备
- 禁止使用药水和喷溅型药水
- 禁止使用末影珍珠

## 比赛规则

### 1v1对决

1. 双方同意后才能开始
2. 使用标准装备或协商装备
3. 管理员监督下进行
4. 败方不得要求再战

### 团队战

1. 双方人数必须对等
2. 提前确定规则和地图
3. 提供观战频道
4. 胜方获得奖金和积分

## 积分系统

参与PVP可获得积分:

- 1v1胜利: +10分
- 团队战胜利: +15分
- 参与奖: +5分

积分可用于兑换稀有物品和称号。

## 举报违规

遭遇PVP违规:

1. 截图或录像证据
2. 使用 `/report <玩家名> <原因>`
3. 联系Discord管理团队

核实后将对违规者进行处罚。
