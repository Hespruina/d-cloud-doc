# 建筑规范

详细的建筑规则和指南。

## 建筑限制

### 允许内容

- 所有原版材料均可使用
- 允许红石电路和自动化装置
- 允许地下建筑
- 允许水上建筑

### 限制内容

- 高频红石装置(每秒超过20次操作)
- 大规模刷怪塔(影响服务器性能)
- 超大型农场(超过100x100范围)
- 实体聚集装置(超过50个实体)

### 禁止内容

- 服务器卡顿装置(故意降低TPS)
- 区块加载器(超过10个区块)
- 恶意TNT陷阱
- 隐藏式偷窃装置

## 公共建筑指南

### 出生点周边(0-500格)

- 保持简洁美观
- 符合服务器主题风格
- 需要管理员审批
- 提供基础功能(商店、信息等)

### 道路连接

- 主道路宽度建议5-7格
- 使用统一的路面材料
- 提供照明和路标
- 不要阻断现有道路

## 私人领地建筑

### 建议

- 保持建筑风格协调
- 避免过于密集的建筑
- 预留扩展空间
- 做好内部装饰

### 领地边界

- 可以使用围栏标记边界
- 不要紧贴他人领地建造
- 保持至少5格缓冲距离

## 特殊建筑

### 商店建筑

- 提供清晰的价格标识
- 保持货架整齐美观
- 定期更新商品

### 农场

- 控制农场规模
- 使用高效而非大体积设计
- 避免使用实体聚集

### PVP竞技场

- 需要管理员审批
- 保证对称性和公平性
- 提供观战区域
