# 经济系统

服务器提供完善的经济系统,让玩家能够自由交易和互动。

## 货币说明

服务器使用金币作为基础货币单位。

### 获取金币

- 出售物品给系统商店
- 玩家间交易
- 参与活动获得奖励
- 挖矿和种植

### 初始资金

新玩家注册后自动获得 500 金币启动资金。

## 玩家交易

### 直接转账

```
/pay <玩家名> <金额>
```

示例:
```
/pay Steve 100
```

### 面对面交易

使用交易界面进行安全交换:

```
/trade <玩家名>
```

双方放入物品后确认即可完成交易。

## 商店系统

### 创建商店

```
/shop create <商店名称> <buy/sell> <价格> <数量>
```

参数说明:
- `buy`: 系统从玩家收购物品
- `sell`: 系统向玩家出售物品

示例:
```
/shop create Diamond sell 100 1
```

### 商店管理

```
/shop add <物品> <价格> <数量>    # 添加商品
/shop remove <物品>               # 移除商品
/shop info                        # 查看商店信息
```

## 系统商店

系统提供基础物品的收购和出售服务。

### 收购价格

| 物品 | 价格 |
|------|------|
| 钻石 | 100/个 |
| 铁锭 | 20/个 |
| 金锭 | 30/个 |
| 绿宝石 | 50/个 |
| 下界石英 | 15/个 |

### 出售价格

系统商店出售价格通常为收购价格的 1.5 倍。

## 经济规则

1. 禁止利用漏洞刷取金币
2. 禁止使用外挂自动化交易
3. 允许玩家间自由交易
4. 大额交易建议当面进行避免诈骗
