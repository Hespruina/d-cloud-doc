# 服务器特色

D-Cloud Minecraft 服务器致力于提供最优质的生存体验,以下是我们的核心特色。

## 纯净生存

我们保持最接近原版Minecraft的游戏体验:

- 无多余修改,保持游戏平衡
- 原版合成表、方块、生物全部保留
- 只添加必要的辅助插件(领地、经济、传送)

## 社区驱动

服务器由社区共同建设和维护:

- 玩家投票决定服务器更新方向
- 定期举办社区会议收集建议
- 活跃的管理团队及时响应问题

## 长期运营

我们致力于长期稳定运营:

- 服务器地图永不重置
- 定期备份玩家数据
- 持续优化服务器性能

## 公平游戏环境

- 完善的反作弊系统
- 严格处理外挂和作弊行为
- 所有玩家平等对待

## 玩家权益

### 领地保护

每个玩家都可以保护自己的领地:

- 默认领地大小: 100x100
- 可通过游戏时间扩展
- 完全控制谁能进入

### 经济平衡

- 基础启动资金: 500金币
- 玩家间自由交易
- 系统商店收购常见物品
- 稳定物价,避免通货膨胀

## 特色玩法

### 每周活动

| 活动类型 | 时间 | 奖励 |
|---------|------|------|
| 建筑比赛 | 每周六 20:00 | 1000-5000金币 |
| PVP竞技 | 每周日 20:00 | 稀有道具 |
| 寻宝活动 | 随机时间 | 特殊称号 |

### 成就系统

完成特定目标可获得成就和奖励:

- 首次加入服务器: 100金币
- 创建第一个领地: 200金币
- 累计在线100小时: 特殊称号
- 参与10次活动: 稀有粒子效果

### 玩家商店

你可以开设自己的商店:

```
/shop create <名称> <买/卖> <价格> <数量>
```

允许其他玩家直接从你的箱子购买物品。

## 服务器规则概要

详细的规则请查看 [服务器规则](/rules/main-rules) 页面,以下是核心要点:

1. 禁止使用任何作弊客户端或模组
2. 尊重其他玩家,禁止辱骂和骚扰
3. 禁止恶意破坏他人建筑
4. 禁止在公共区域建造不适当内容
5. 遵守领地规则,不要跨越他人领地边界
