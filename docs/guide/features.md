# 服务器特色

D-Cloud Minecraft 服务器致力于提供最优质的生存体验,以下是我们的核心特色。

## 纯净生存

我们保持最接近原版Minecraft的游戏体验:

- 原版合成表、方块、生物全部保留
- 只添加必要的辅助插件(领地、经济、传送)

## 社区驱动

服务器由社区共同建设和维护:

- 玩家群体共同决定服务器更新方向
- 活跃的管理团队及时响应问题

## 长期运营

我们致力于长期稳定运营:

- 服务器地图永不重置
- 定期备份玩家数据
- 持续优化服务器性能

## 每日签到

服务器提供每日签到功能，鼓励玩家保持活跃并获取奖励。

### 如何签到

在主城找到签到NPC或输入命令 `/signin gui` 即可打开签到界面。

![签到NPC](/npcs/signin.png)

### 基础签到奖励

每日正常签到可获得以下奖励：
- **钻石** x1
- **金锭** x3
- **铁锭** x6
- **雨点（游戏币）** x100

### 特殊日期奖励

在特定节日签到可获得额外奖励：
- **元旦 (01-01)**: 钻石 x10 + 钻石块 x1
- **国庆节 (10-01)**: 绿宝石 x5
- **圣诞节 (12-25)**: 苹果 x64

### 特殊时段奖励

在特定时段签到可获得特别关怀：
- **晚间签到 (22:30后)**: 牛奶桶 x1（提醒早点休息）
- **凌晨签到 (03:00前)**: 牛奶桶 x1（提醒早点休息）

### 连续签到奖励

- **月度连续签到15天**: 在1-3月期间，奖励翻倍
- **月度累计签到15天**: 在1-3月期间，额外获得游戏币 x1000

### 排名奖励

- **每日首位签到**: 所有奖励翻倍（钻石 x2、金锭 x6、铁锭 x12）

## 每日收购

服务器提供每日收购功能，每天0点刷新收购物品列表，玩家可以出售~~一些破烂~~各种物品获取雨点（游戏币）。

在主城找到每日收购NPC，右键打开收购菜单即可进行交易。

![每日收购NPC](/npcs/dailybuyshop.png)

![收购菜单](/menus/dailybuyshop.png)

## 回收站系统

服务器提供回收站功能，帮助玩家找回意外丢失的物品。

### 如何打开回收站

在主城找到回收站NPC或使用命令打开回收站菜单。

![回收站主菜单](/menus/mainmenu-wtc.png)

### 全服垃圾桶

全服垃圾桶可以当作公共箱子使用，存放清理插件定期清理的物品。

![全服垃圾桶](/menus/wtc-world.png)

### 个人垃圾桶

个人垃圾桶是私人空间，用于存放你不小心丢掉的物品：

- 物品丢到地面一段时间没有拾取，会自动收入个人垃圾桶
- 仙人掌、岩浆吞掉的物品可以在个人垃圾桶找回

![个人垃圾桶](/menus/wtc-player.png)

**注意**：个人垃圾桶中的物品会自动清理，请及时取出。

#### 赎回物品

从个人垃圾桶中赎回物品需要支付一定的游戏币。

![赎回物品示例](/example/pay-for-wtc-player.png)
